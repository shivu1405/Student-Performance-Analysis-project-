# Student Performance Analysis

An exploratory data analysis project built using Python (pandas, matplotlib).

## What this project covers
- Impact of test preparation courses on exam scores
- Relationship between parental education level and student performance
- Gender-based score differences across math, reading, and writing

## Key Findings
- Students who completed prep courses scored up to 9.9 points higher
- Parental education shows a clear positive correlation with student scores
- Female students outperform in reading and writing; males lead in math

## Tools Used
Python · pandas · matplotlib · Google Colab
